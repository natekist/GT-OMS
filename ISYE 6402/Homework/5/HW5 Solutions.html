Assuming that a time series does not present serial correlation, if instead there is significant serial correlation in the squared values of the time series, then heteroskedastic modelling techniques could be used to model this serial correlation.

Hint: One example you could think of is the return stock price, which often shows no serial correlation in the time series itself. We have seen this in some of the examples in Module 3.

4673
 
True
 
exact_answer none 8853837 4673
margin of error +/-
338
 
False
 
exact_answer none 8853837 338
margin of error +/-

We generally model the expectated behavior of a time series using ARMA modeling, but if there is no serial correlation in the first moment or the expectation, then we don't need to apply any ARMA modeling. This is common in the analysis of return stock prices for example. However, as we have seen in such applications, it is possible to have volatility, which is reflected as systematic changes in variance over time, which can be identified as serial correlation in the squared time series data. Hence we can apply heteroskedastic modeling.

A t-interval is derived using the classical formula:

(mean-t-qunatile*std,mean+t-qunatile*std)

In this example, we have the followings:

mean = 0.8891 (corresponding to the b1 estimated coefficient)

std = 0.01652 (corresponding to the b1 estimated standard deviation)

t-quantile for  95% confidence level is the t-quantile at 0.05/2. However, this example does not provide the degrees of freedom to plug in this formula and instead we are using the z-quantile, which is approximately equal to 1.96.

This will give us the following interval: (0.8567, 0.9215).
